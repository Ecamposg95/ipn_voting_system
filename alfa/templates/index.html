Ingeniería en Software

Proyecto de Reconocimiento Facial

Este proyecto fue desarrollado como parte de la materia de Ingeniería en Software en la ESIME Zacatenco del Instituto Politécnico Nacional. La aplicación permite a los usuarios registrarse y autenticarse utilizando sus rostros.

El proyecto está construido con Flask y JavaScript, utilizando la biblioteca face_recognition para el procesamiento de imágenes.
